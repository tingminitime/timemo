# GitHub Copilot 介紹與心得

[GitHub Copilot](https://github.com/features/copilot) 是由 GitHub 和 OpenAI 合作面向開發者的 AI 工具，採用 OpenAI 的 Codex 模型，這個模型由大量的開源文本、原始碼訓練，其中也包含 GitHub 上公開的項目，驅動 Visual Studio Code、Visual Studio、Vim、JetBrains 等開發工具中使用。

GitHub Copilot 會在開發者的開發環境中，讀取上下文即時建議程式碼片段，很大程度上可以幫助開發者縮短撰寫重複性程式碼的時間，讓開發者專注在架構的規劃、撰寫更重要的邏輯上。
